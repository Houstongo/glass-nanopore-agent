# Etching Knowledge Base Seed

这个文件夹是为“玻璃纳米孔电刻蚀智能体”准备的知识库种子目录，按知识库建设的四个核心维度整理：

- `01_theory_and_literature`
  - 放论文、学位论文等理论与工艺经验材料
- `02_sop_and_workflow`
  - 放 SOP、流程规范、实验操作说明
- `03_agent_and_rag`
  - 放智能体、RAG、知识图谱、硬件桥接相关代码
- `04_hardware_and_firmware`
  - 放 STM32 固件、回调、串口/ADC/TIM 控制和 ESP8266 边缘控制文件
- `05_experiment_data`
  - 放已整理的实验数据库与导入脚本

## 选择原则

本次归档优先选择：

- 能直接作为 RAG 文本源的文档
- 能解释 Agent 如何检索、推理和调用工具的源码
- 能说明底层硬件原语、ADC/串口/定时器行为的固件文件
- 已经结构化完成的实验数据库文件

## 为什么没有整包复制 `cvdata`

`cvdata` 目录体量较大，且包含大量原始图片与分组目录。当前先放入已经整理好的：

- `etching_experiments.sqlite`
- `etching_experiment_db.py`

这样后续可以先基于数据库做筛选，再按需要把高价值实验图片批量补充进知识库，而不是盲目复制所有原始文件。

如果下一步要扩充，我建议优先补：

1. 过去三年成功率最高的 50 组实验图片目录
2. 带有波形、参数、图片三者同时存在的实验批次
3. 异常案例对应的 SEM/原图目录
