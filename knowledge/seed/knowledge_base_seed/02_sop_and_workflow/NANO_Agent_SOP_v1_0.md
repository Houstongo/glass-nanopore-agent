# NANO-Agent v1.0 智能玻璃纳米刻蚀平台规范说明书与标准作业程序 (SOP)

## 第一部分：智能玻璃纳米刻蚀平台 (NANO-Agent v1.0) 规范说明书

### 1. 系统定义
NANO-Agent v1.0 是一款基于**大小模型协同架构**的具身智能制备平台，专用于玻璃纳米通道及金属纳米尖端的自动化电化学刻蚀。系统通过 **MCP (Model Context Protocol)** 协议将底层物理动作映射为高层语义原语，实现从实验意图到硬件执行的端到端自动化。

### 2. 三层递阶架构规范
*   **战略层 (L2 - 中央服务器)**：运行基于 FastAPI 的智能代理，负责 RAG 知识检索、Coze 风格工作流编排、策略重构及长短期记忆管理。
*   **感知层 (L1 - 边缘网关/ESP8266)**：负责 115200bps 高速异步数据流转发，并对 100Hz 采样电流进行轻量级边缘计算，提取异常特征。
*   **执行层 (L0 - STM32 控制器)**：负责 PWM 步进电机（Z轴位移/旋转轴）的精密驱动、继电器逻辑切换及硬件级安全截断。

### 3. 核心技术参数
*   **位移精度**：$2.5\mu\text{m}$ / 脉冲 (基于 $400$ 脉冲/转驱动器)。
*   **采样频率**：$100\text{Hz} - 1\text{kHz}$ (ADC2 中断驱动)。
*   **通讯延迟**：$\text{PC} \leftrightarrow \text{Device} < 50\text{ms}$。
*   **调控模式**：支持恒压 (DC) 及交流 (AC) 脉冲模式，支持波形发生脚本动态注入。

## 第二部分：标准作业程序 (SOP)

### 阶段一：实验前置准备与系统自检
1.  **环境标定**：确认电解液（如 $CaCl_2$）浓度与知识库 $K$ 中的预设值对齐，并记录当前温湿度。
2.  **网络握手**：启动 Lab-OS 中央控制台，检查并确保 ESP8266 与 PC 的 WebSocket 通讯链路处于“Active”状态。
3.  **气密性自检 (Critical Check)**：
    *   调用 `Seal_Check_Tool`，系统自动对电解液腔室加压。
    *   Agent 实时监测压力衰减曲线。
    *   **判定标准**：若衰减系数 $k > \text{Threshold}$，Agent 将锁定后续执行序列，提醒手动检查密封圈。

### 阶段二：工作流编排与参数初始化
1.  **任务定义**：在 GUI 界面中拖入“自动化刻蚀”任务节点。
2.  **知识库注入**：Agent 根据目标直径与材料类型，自动从 RAG 知识库提取初始工艺参数向量 $\theta_0 = [V, f, d_{imm}]$。
3.  **报警阈值预置**：将异常检测灵敏度参数 $\alpha, \beta$ 下发至感知层时序监控模块。

### 阶段三：自动化制备逻辑执行 (Flag 16 序列)
1.  **液面捕捉**：STM32 驱动 Motor 1 以慢速上升，ADC2 实时侦测电流 $I(t)$。
2.  **触碰自锁**：一旦捕捉到突变信号（$I > \text{Threshold}$），系统立即产生 `Contact_Detected` 信号并启动位移清零。
3.  **定量浸入**：基于 `CalPulseTime` 函数，系统将铂丝精准送入液面以下预设的 $d_{imm}$ 深度。
4.  **闭环调控**：波形发生器执行脚本定义的脉冲流，若 Agent 识别到 $dI/dt$ 异常，将动态改写脚本以微调电压步长 $\delta V$。
5.  **终点截断**：侦测到断裂点脉冲后，STM32 需在毫秒级执行继电器 LK2 关断，以保持尖端几何形貌的锐利度。

### 阶段四：后处理、质量评价与知识闭环
1.  **自动复位**：Motor 1 执行回落保护动作，Motor 2 将样品旋转至指定接收位。
2.  **图像表征**：调用视觉计算工具自动计算**曲率半径**与**锥角**。
3.  **质量分级**：CNN 模型对成膜质量进行优/良/差分级评分。
4.  **复盘与对齐**：将全流程数据（波形、图像、环境参数）存入数据库。Agent 进行实验复盘，根据结果偏差调整优化器 $J(\theta)$ 的权重，实现实验知识的持续迭代。

## 附录：刻蚀异常诊断表 (Troubleshooting Table)

| 异常现象 | dI/dt 特征 | Agent 归因 (RAG 支持) | 处理动作 |
| :--- | :--- | :--- | :--- |
| **电流剧烈震荡** | 高频随机毛刺 | 气泡附着或局部极化 | 执行 Motor 2 微动或下发脉冲清洗脚本 |
| **电流缓慢漂移** | 趋势性单调偏移 | 电解液蒸发导致浓度漂移 | 调取密封自检，微调 Vstep |
| **无预警截断** | 阶跃式归零 | 机械碰撞或接触不良 | 触发“实验报废”逻辑，记录失效模式 |
